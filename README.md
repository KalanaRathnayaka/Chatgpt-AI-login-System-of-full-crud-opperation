# Full Stack Authentication Management System

## Overview

A full-stack authentication system developed using Node.js, Express.js, MongoDB, and JavaScript.

## Features

- User Registration
- User Login
- JWT Authentication
- Password Hashing
- MongoDB Integration
- Protected Routes
- REST API Architecture

## Technologies

- Node.js
- Express.js
- MongoDB
- JWT
- JavaScript
- HTML
- CSS

## Installation

npm install

## Run

npm start
