# Full Stack Authentication Management System

## Overview

 AI-Assisted Full Stack Login System Comparison

This project was created to compare full-stack login system implementations generated using ChatGPT and Gemini from the same prompt.

The purpose of this project was to evaluate code structure, functionality, authentication flow, and overall development quality between both AI-generated approaches.

A full-stack authentication system developed using Node.js, Express.js, MongoDB, and JavaScript.

## Features

- User Registration
- User Login
- JWT Authentication
- Password Hashing
- MongoDB Integration
- Protected Routes
- REST API Architecture

## Technologies

- Node.js
- Express.js
- MongoDB
- JWT
- JavaScript
- HTML
- CSS
